# 芯联键合上位机

这是一个 .NET 8 WinForms 上位机骨架，包含：

- PLC Modbus TCP 扫码触发监听与人工处理信号回写
- EAP SECS/GEM 参数配置界面
- 霍尼韦尔扫码枪串口配置、触发指令发送与接收
- PLC 单个产品码触发扫码流程、超时人工确认
- SQLite 本地扫码数据保存和表格显示
- 本地日志记录和日志界面

## 运行

```powershell
dotnet run
```

## 本地文件

运行后会在程序输出目录下生成：

- `config.json`：PLC、EAP、扫码枪配置
- `Data/scans.db`：SQLite 扫码数据
- `Logs/app-yyyyMMdd.log`：本地日志

## PLC 扫码握手

- `D50`：扫码触发寄存器，值为 `1` 时触发扫码。上位机触发后会复位 `D50=0`。
- `D52`：当前扫码位置，只在 `D50=1` 触发后读取一次，程序作为 `PORTID` 保存，取值 `1/2/3/4`。
- `D51`：扫码枪有产品码返回后，上位机写入 `1`。
- `M838`：扫码异常后，UI 点击“重新扫码”写入 `1`。
- `M839`：扫码异常后，UI 手动输入产品码并上传时写入 `1`。
- `M835`：扫码异常后，UI 点击“跳过当前位”写入 `1`。

霍尼韦尔扫码枪触发指令为 `LON + CR`，HEX：`4C 4F 4E 0D`。暂停/关闭指令预留为 `LOFF + CR`，HEX：`4C 4F 46 46 0D`。

## SECS/GEM

当前版本先实现配置界面和保存逻辑。只有扫码成功或人工输入了实际产品码才保存 SQLite 并记录一条只包含产品码的 `S6F11` 上报日志；跳过当前位不会写空产品码记录。真实通讯库可以在后续新增 `EapClient` 类接入，例如连接、Select.req、LinkTest、事件上报等逻辑。

```sml
S6F11 W
<L [3]
  <U4 DATAID>
  <U4 CEID>
  <L [1]
    <L [2]
      <U4 RPTID>
      <L [1]
        <A PRODUCTCODE>
      >
    >
  >
>.
```
